# Xiao Jiang (江筱) - Academic Website

This is the source code for my personal academic website, hosted on GitHub Pages. 

## Features

- **Condensed Matter Physics Theme**: A modern, clean design inspired by physical concepts (crystal lattices, quantum potentials), utilizing a custom color palette (Deep Orbital Blue, Fermion Gray).
- **Pure CSS Interactivity**: High-performance, lightweight design with zero JavaScript dependencies for core functionality.
  - Interactive "View Abstract" toggles using HTML5 `<details>` and `<summary>`.
  - CSS3 Animations for card floating, hover effects, and skill bars.
- **Responsive Design**: Fully responsive layout built on top of [Pico CSS](https://picocss.com/) (v1), ensuring great visibility on all devices.
- **Academic Focused**:
  - **Publications**: Auto-numbered lists with distinctive "First Author" badges and citation styles.
  - **Featured Research**: Highlighted section for the most impactful recent work.
  - **Researcher Card**: A glassmorphic digital business card for contact information.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling (`custom_style.css`) with CSS Variables for theming.
- **Pico CSS**: Minimalist CSS framework for base styling.
- **Icon Libraries**: 
  - [Academicons](https://jpswalsh.github.io/academicons/) (Google Scholar, DOI, etc.)
  - [Font Awesome](https://fontawesome.com/) (General UI icons)

## Usage

This site is designed to be hosted directly on GitHub Pages. 
To modify the content, simply edit the HTML files:
- `index.html`: Homepage (About, Skills).
- `publications.html`: List of publications and research highlights.
- `contact.html`: Contact information.
